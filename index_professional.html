02

Méthode des rectangles droits

Approche numérique classique où la hauteur de chaque rectangle est la valeur de la fonction en l'extrémité droite du sous-intervalle.

Formule de quadrature
\[y_{\text{num}}(x)\;\approx\; h\sum_{k=1}^{N}f(t_k,\,x),\qquad t_k = k\,h,\quad h=\frac{1}{N}\]
Forme compacte pour l'implémentation
\[\int_{0}^{1}f(t,x)\,dt\;\approx\;\frac{1}{N}\sum_{k=1}^{N}f\!\left(\frac{k}{N},\,x\right)\]
Fonction sous-intégrande
\[f(t,\,x)\;=\;\sqrt{\frac{x-t}{x+t}},\qquad x\in[1;3],\quad t\in[0;1]\]

Déroulement de l'algorithme

①

Génération des points \(x_i\)

21 valeurs sur \([1;3]\) avec step = 0.1

②

Calcul du pas \(h = 1/N\)

Pour chaque \(N\in\{5,10,20,50\}\), le pas d'intégration sur \([0;1]\)

③

Accumulation de la somme

Pour \(k=1\ldots N\) : évaluer \(f(k \cdot h, x)\) et accumuler

④

Multiplication par \(h\)

Résultat : \(y_{\text{num}} = h \times\) somme

⑤

Calcul de l'erreur

Retenir le max de \(|y_{\text{num}}-y_{\text{exact}}|\) sur tous les \(x_i\)

03

Solution analytique exacte

Par la substitution \(u=\sqrt{(x-t)/(x+t)}\), on transforme l'intégrale en arctangente et fonctions algébriques, puis on évalue aux bornes transformées.

1

Substitution

On pose \(u=\sqrt{\dfrac{x-t}{x+t}}\), donc \(u^2=\dfrac{x-t}{x+t}\). En isolant \(t\) :

\[t \;=\; x\,\frac{1-u^2}{1+u^2}\]
2

Différentielle de \(t\)

Règle du quotient appliquée à \(t(u)\) :

\[dt \;=\; x\cdot\frac{-4u}{(1+u^2)^2}\,du\]
3

Changement de bornes

\(t=0\;\Rightarrow\;u=1\)  |  \(t=1\;\Rightarrow\;u=\sqrt{\dfrac{x-1}{x+1}}\)

\[y(x)\;=\;-4x\int_{1}^{\sqrt{(x-1)/(x+1)}}\frac{u^2}{(1+u^2)^2}\,du\]
4

Décomposition et intégrales tabulaires

Utilisant \(\dfrac{u^2}{(1+u^2)^2}=\dfrac{1}{1+u^2}-\dfrac{1}{(1+u^2)^2}\) :

\[\int\frac{u^2}{(1+u^2)^2}\,du\;=\;\frac{1}{2}\!\left(\arctan u-\frac{u}{1+u^2}\right)+C\]
5

Évaluation aux bornes

Avec \(U=\sqrt{\dfrac{x-1}{x+1}}\) et \(1+U^2=\dfrac{2x}{x+1}\) :

\[\frac{U}{1+U^2}\;=\;\frac{\sqrt{x^2-1}}{2x}\]
✦ Résultat analytique final
\[y(x)\;=\;-2x\,\arctan\!\sqrt{\dfrac{x-1}{x+1}}\;+\;\sqrt{x^2-1}\;+\;x\!\left(\dfrac{\pi}{2}-1\right)\]

// Fonction sous-intégrande
double f(double t, double x) {
    return sqrt((x - t) / (x + t));
}

// Méthode des rectangles droits
double numIntegration(int N, double x) {
    double h = 1.0 / N;
    double sum = 0.0;
    for (int k = 1; k <= N; k++) {
        sum += f(k * h, x);
    }
    return h * sum;
}

// Solution analytique
double exactSolution(double x) {
    double u = sqrt((x - 1.0) / (x + 1.0));
    return -2.0 * x * atan(u) + sqrt(x*x - 1.0) 
           + x * (M_PI / 2.0 - 1.0);
}

Conclusion

Convergence Numérique

La méthode des rectangles droits converge vers la solution exacte.

Stabilité Computationnelle

Pour x ∈ [1;3], l'algorithme reste stable.

Performance Asymptotique

Δmax ~ O(1/N) : l'erreur décroît linéairement.
