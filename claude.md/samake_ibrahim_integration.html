02

Méthode des rectangles droits

La fonction sous l'intégrale ne s'intègre pas en fonctions élémentaires sur tout le domaine. On applique donc la méthode des rectangles droits, remplaçant chaque sous-intervalle par la valeur de la fonction en son extrémité droite.

Formule de quadrature — Presmoy Method
y_num(x) ≈ h · Σ f(tₖ, x) [k = 1 … N]

où tₖ = k·h, h = (b−a)/N = 1/N

f(t, x) = √( (x−t) / (x+t) )
Forme compacte pour l'implémentation
∫₀¹ f(t,x) dt ≈ (1/N) · Σ f(k/N, x) [k = 1 … N]

Déroulement de l'algorithme

①

Génération des points xᵢ

21 valeurs uniformément réparties sur [1 ; 3] avec pas step = (d−c)/20 = 0.1

②

Calcul du pas h

Pour chaque N, on calcule h = 1/N — le pas d'intégration sur [0 ; 1]

③

Évaluation de la somme

Pour k = 1..N, on évalue f(k·h, x) et on accumule la somme

④

Multiplication par h

Le résultat final est y_num = h × somme, approximant l'aire sous la courbe

⑤

Calcul de l'erreur

On compare avec la solution exacte : |y_num − y_exact|, et on retient le max sur tous les xᵢ

03

Solution analytique

Par substitution u = √((x−t)/(x+t)), on transforme l'intégrale en une combinaison d'arctangente et de fonctions algébriques, puis on évalue aux bornes.

1

Substitution

On pose u = √((x−t)/(x+t)), donc u² = (x−t)/(x+t), d'où :

t = x · (1 − u²) / (1 + u²)
2

Différentielle

En dérivant t par rapport à u par la règle du quotient :

dt = x · (−4u) / (1 + u²)² du
3

Changement de bornes

t = 0 → u = 1  |  t = 1 → u = √((x−1)/(x+1))

y(x) = −4x · ∫₁^√((x−1)/(x+1)) u² / (1+u²)² du
4

Intégration par décomposition

On utilise u²/(1+u²)² = 1/(1+u²) − 1/(1+u²)² et les intégrales tabulaires :

I = ½ ( arctan(u) − u/(1+u²) ) + C
5

Évaluation aux bornes

En posant U = √((x−1)/(x+1)) et en notant que 1+U² = 2x/(x+1) :

U / (1+U²) = √(x²−1) / (2x)
Résultat analytique final
y(x) = −2x · arctan( √((x−1)/(x+1)) ) + √(x²−1) + x · (π/2 − 1)

Remplacement point→virgule

// ── Constante π ───────────────────────────────
const double PI = acos(-1.0);

// ── Fonction sous-intégrande ──────────────────
double f(double t, double x) {
    double num = x - t,  deno = x + t;
    if (deno <= 0) return 0.0;
    return sqrt(num / deno);
}

// ── Méthode des rectangles droits ────────────
double rectangle_droite(double a, double b,
                          int N, double x) {
    double sum = 0.0,  h = (b - a) / N;
    for (int k = 1; k <= N; k++)
        sum += f(a + k * h, x);
    return h * sum;
}

// ── Solution analytique exacte ────────────────
double y_analy(double x) {
    double u = sqrt((x-1) / (x+1));
    return -2.0*x*atan(u)
           + sqrt(x*x - 1)
           + x*(PI/2 - 1);
}

Astuce Excel — remplacement point → virgule

La version russe d'Excel utilise la virgule comme séparateur décimal. Après écriture du fichier resultat.csv, le programme relit tout le contenu et remplace chaque '.' par ',', puis sauvegarde sous resultat_final.csv — compatible Excel sans configuration.

Convergence en O(h)

Quand N double, l'erreur est divisée par ≈ 2. La méthode des rectangles droits est d'ordre 1 — cohérent avec la théorie.

Précision vs coût

N = 50 donne Δmax ≈ 0.0104, soit une précision au centième. Pour des applications pratiques, N = 20 est un bon compromis.

Valeur de l'analytique

L'expression exacte permet de valider le code et de mesurer rigoureusement l'erreur pour chaque N, démontrant la fiabilité du calcul.
